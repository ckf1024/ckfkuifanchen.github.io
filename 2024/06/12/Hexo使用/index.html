Hexo使用

记录如何使用 Hexo 发布个人博客，Markdown语法等，主要为记录源文件格式。

一、Hexo基本操作

1.1 建站

指定文件夹中新建立网站:hexo init <folder>

进入新建文件夹：cd <folder>

安装依赖：npm install

1.2 创建新的md文件

创建新的文章：hexo new [post] <title>

创建新的草稿：hexo new [draft] <title>

创建新的页面：hexo new [page] <title>

1.3 其他常用命令

部署到GitHub：hexo g -d

启动本地服务器：hexo s

清理缓存和已生成的静态文件（public）：hexo clean
